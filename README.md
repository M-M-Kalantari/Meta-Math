# MetaMath

**MetaMath** is a multifunctional Android calculator app that goes beyond basic calculations. It includes unit conversion, financial tools, geometry calculations, number base conversion, mathematical formulas, and a BMI calculator. Built using Java, XML, and SQLite with a focus on practical real-world usage.  

[Download on Myket](https://myket.ir/app/persian.googolplexian.metamath)

# متا مَت

**MetaMath** یک ماشین‌حساب حرفه‌ای و چندمنظوره برای اندروید است که فراتر از محاسبات ساده عمل می‌کند.  
این برنامه شامل تبدیل واحد، ابزارهای مالی، محاسبات هندسی، تبدیل مبنا، فرمول‌های ریاضی و محاسبه BMI می‌باشد. با استفاده از **Java، XML و SQLite** توسعه داده شده و روی کاربردی بودن تمرکز دارد.  

[دانلود از مایکت](https://myket.ir/app/persian.googolplexian.metamath)

## Key Features / قابلیت‌ها

- Advanced calculator / ماشین‌حساب پیشرفته  
- Unit conversion / تبدیل واحد (length, weight, temperature…)  
- Financial calculations / محاسبات مالی  
- Geometry calculations / محاسبات هندسی (areas, volumes, shapes)  
- Number base conversion / تبدیل مبنا (binary, decimal, etc.)  
- Mathematical formulas / فرمول‌های ریاضی  
- BMI calculator / محاسبه BMI  
- Local data handling with SQLite / مدیریت داده‌ها به‌صورت محلی با SQLite
